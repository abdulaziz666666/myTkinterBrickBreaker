  <li>تثبيت مدمر المستطيلات.exe</li>
</ul>

<h2>التثبيت للمطورين</h2>
<ul>
إذا أردت تدقيق ملف اللعبة فنزل جميع الملفات المرفقة. مع العلم أنك وإن لم تنزل إلا الملف البرمجي فسيعمل بلا أخطاء برمجية ولكن مع نقص في الميزات: <br>
  <li>أيقونة اللعبة وشعارها الظاهر في الصفحة الرئيسية لن يكونا ظاهرين</li>
  <li>لن يكون هناك أي مؤثرات صوتية</li>
</ul>
<br>
<ul>

إذا نزلت جميع المرفقات فضعهم في نفس المجلد وفك الضغط عن مجلد المؤثرات الصوتية:
  <li>sfx</li>
وضعه في نفس المجلد الذي يحوي باقي الملفات <br>
</ul>
